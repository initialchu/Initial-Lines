# 初笺

# 一、项目背景

开发一个辅助编辑markdown文件的桌面应用，用户可以导入自己的.md文件进行编辑，也可以在选择保存创建一个文件（保存在用户）。这个软件在启动时没有弹窗（用户下载后第一次启动会有），后面图标将在任务栏被隐藏

右键点击后可以“主页面”唤起软件窗口。默任情况下，点击ctrl+空格能唤起一个悬浮的文本编辑窗口。

软件本体主页面有一个文本编辑窗口，可以选择编辑.md文件，或纯文本文件（默认.md），点击可以切换渲染预览。

- 注：这是初代构想，后面会再修改。
